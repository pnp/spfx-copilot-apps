# Support

## How to file issues and get help

This project uses GitHub Issues to track bugs and feature requests for the samples in this repository. Please search the [existing issues](https://github.com/pnp/spfx-copilot-apps/issues) before filing new issues to avoid duplicates. For a new issue, [open a new issue](https://github.com/pnp/spfx-copilot-apps/issues/new).

The samples in this repository are provided _as is_ and are maintained as an open source community effort. They are **not** officially supported Microsoft products and do not come with a service-level agreement (SLA). The community, including the maintainers and contributors, does its best to help resolve issues, but assistance is provided on a best-effort, volunteer basis.

For help and questions about using this project, please:

- Review the `README.md` file inside the specific sample folder you are working with.
- Search [existing issues](https://github.com/pnp/spfx-copilot-apps/issues) and [discussions](https://github.com/pnp/spfx-copilot-apps/discussions) to see if your question has already been answered.
- Open a new issue or discussion if you cannot find an answer.

For general SharePoint Framework or SharePoint Copilot questions that are not specific to a sample, use the [sp-dev-docs repository issue list](https://github.com/SharePoint/sp-dev-docs/issues).

## Microsoft Support Policy

Support for the samples in this repository is limited to the resources listed above.
